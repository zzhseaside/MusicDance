# MusicDance
让音乐的节奏律动表达出来

如图
![img.png](img.png)

工具是 CocosCreator 3.8.0

实现思路:
1. 获取音频播放器的音频采样数据,
2. 用采样数据和shader把每列的数据用色值进行过度一下
